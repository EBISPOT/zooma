Zooma jQuery Plugin

In this page you can find some examples of the Zooma plugin. At the end of the page you will find the link to the files

Examples:

Simple Setup:

A basic Zooma widget, with autocomplete and tagging boxes generated by Zooma within a div element:

Code:
<!DOCTYPE html> <html> <head> <link rel="stylesheet" href="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.css"> <link rel="stylesheet" href="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.css"> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.min.js"></script> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.min.js"></script> </head> <body onload="$('#zooma-search').zooma();"> <div id="zooma-search" style="width: 500px;" /> </body> </html>
Simple Setup on an input box:

A basic Zooma widget, with autocomplete and tagging embedded into a single input element:

Code:
<!DOCTYPE html> <html> <head> <link rel="stylesheet" href="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.css"> <link rel="stylesheet" href="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.css"> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.min.js"></script> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.min.js"></script> </head> <body onload="$('#zooma-search').zooma();"> <label for="zooma_search">Enter your term:</label> <input type="text" id="zooma-search" name="zooma-search" style="width: 400px" placeholder="Zooma-enabled term"/> </body> </html>
Advanced setup using separated autocomplete/suggestion boxes

A slightly more advanced usecase for Zooma, where you want to separately define an input element for autocomplete and a select box for tags.

Code:
<!DOCTYPE html> <html> <head> <link rel="stylesheet" href="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.css"> <link rel="stylesheet" href="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.css"> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.min.js"></script> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.min.js"></script> </head> <body onload="$('#zooma-separated-search').zooma('autocomplete'); $('#zooma-separated-suggestions').zooma('annotate', {'annotation_source': $('#zooma-separated-search')};);"> <label for="zooma_search">Enter your term:</label> <input type="text" id="zooma-separated-search" name="zooma-search" style="width: 400px" placeholder="Zooma-enabled term"/> <label for="zooma-suggestions"></label> <select id="zooma-separated-suggestions" name="zooma-suggestions" multiple></select> </body> </html>

Filtering results:

Zooma let the user also set a filter for the datasources he wants to use and he prefer. This way, users will be able to limit the results to just those sources they want

Here a list of the current available datasources in zooma:

Name URI

Moreover, is also possible to request a specific type of annotation. Zooma will try to favor those annotations with the specific property type. In any case, if no ontology is available for the type or better mapping are available for the requested term, Zooma will always favor the requested term, over the requested type (i.e. a request for 'Fibroblast' with type 'disease' will return the ontology simply for 'Fibroblast');

<!DOCTYPE html> <html> <head> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/vendor.min.js"> </script> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/webjars/zooma-js/2.0.0/zooma.min.js"> </script> </head> <body onload="$('#zooma-datasources-search').zooma('autocomplete'); $('#zooma-datasources-suggestions').zooma('annotate', { 'annotation_source': $('#zooma-separated-search'), 'zooma_datasources': ['efo','ebisc'], 'zooma_preferred_datasources': ['ebisc'], 'zooma_property_type': 'disease' });"> <label for="zooma_search">Enter your term:</label> <input type="text" id="zooma-datasources-search" name="zooma-search" style="width: 400px" placeholder="Zooma-enabled term"/> <label for="zooma-suggestions"></label> <select id="zooma-datasources-suggestions" name="zooma-suggestions" multiple></select> </body> </html>
