REST API Documentation

Introduction

This page describes how to develop against the ZOOMA REST API to search for and retrieve ZOOMA objects.
<zooma.api> refers to the URL of the zooma service you are querying - use http://www.ebi.ac.uk/spot/zooma/v2/api to use the EBI zooma service.

Annotation Summaries

You can search for annotation summary objects in ZOOMA given a property value (and optionally a property type) or retrieve annotation summaries by ID if you know it

Search Example:

To search for annotation summaries about "cancer"
<zooma.api>/summaries/search?query=cancer

Response:
{ result: [ { name: "cancer", mid: "44229DD38831FABF7607132EDB3BAD64893A611D", score: "38.44594", notable: { name: "disease_state; EFO_0000311", id: "summary_from_disease_state_to_EFO_0000311" } }, { name: "cancer", mid: "42CC496DEE019A29717FF9F18077B4B83EE1D728", score: "38.44149", notable: { name: "cell_type; EFO_0000311", id: "summary_from_cell_type_to_EFO_0000311" } }, { name: "cancer", mid: "076362CAE3B9B4078CC3CF40A8CE18F8C3A37EAB", score: "37.439922", notable: { name: "DiseaseState; EFO_0000292", id: "summary_from_DiseaseState_to_EFO_0000292" } } ], status: "/api/status/ok" }

This example clearly shows that there are three types of cancer annotation summary (disease_state: cancer; cell_type: cancer; and DiseaseState: cancer), two of which map to EFO_000311 and one of which maps to EF)_0000292

Retrieve Example:

To retrieve annotation summary '44229DD38831FABF7607132EDB3BAD64893A611D'
<zooma.api>/summaries/44229DD38831FABF7607132EDB3BAD64893A611D
Response:
{ id: "44229DD38831FABF7607132EDB3BAD64893A611D", semanticTags: [ "http://www.ebi.ac.uk/efo/EFO_0000311" ], annotationSummaryTypeID: "summary_from_disease_state_to_EFO_0000311", annotationSummaryTypeName: "disease_state; EFO_0000311", annotatedPropertyValue: "cancer", annotatedPropertyType: "disease_state", annotationURIs: [ "http://www.ebi.ac.uk/zooma/annotation/ANNO_00095246", "http://www.ebi.ac.uk/zooma/annotation/ANNO_00218829", "http://www.ebi.ac.uk/zooma/annotation/ANNO_00218821", "http://www.ebi.ac.uk/zooma/annotation/ANNO_00023571", "http://www.ebi.ac.uk/zooma/annotation/ANNO_00023506", ... ] }

Annotations

Example:

To retrieve the annotation 'ANNO_00095246'
<zooma.api>/annotations/ANNO_00095246
Response:
{ annotatedBiologicalEntities: [ { name: "GSM636920 1", studies: [ { accession: "E-GEOD-25932", uri: "http://www.ebi.ac.uk/arrayexpress/experiment/E-GEOD-25932" } ], uri: "http://www.ebi.ac.uk/gxa/sample/SAMPLE_00031000" } ], annotatedProperty: { propertyType: "disease_state", propertyValue: "cancer", uri: null }, semanticTags: [ "http://www.ebi.ac.uk/efo/EFO_0000311" ], provenance: { source: "http://www.ebi.ac.uk/zooma/db/ATLAS", creationDate: 1343303275135, evidence: "INFERRED_FROM_CURATION", creator: "ZOOMA" }, uri: "http://www.ebi.ac.uk/zooma/annotation/ANNO_00095246" }

ZOOMA 2 jQuery Plugin

ZOOMA 2 is available as a jQuery plugin: zooma.min.js. Simply include this in your HTML to enable ZOOMA based searches.

Example:

To create a ZOOMA search input box:
<!DOCTYPE html> <html> <head> <script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"> </script> <script type="text/javascript" src="http://www.ebi.ac.uk/spot/zooma/scripts/zooma.min.js"> </script> </head> <body onload="$('#zooma_search').zooma();"> <label for="zooma_search">Find Annotations:</label> <input type="text" id="zooma_search"/> </body> </html>
